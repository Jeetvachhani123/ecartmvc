# eCartMVC - Electronic eCommerce Website

eCartMVC is a full-featured **ASP.NET Core MVC eCommerce web application** designed for selling electronic products online.
The system includes both **Admin Panel** and **Customer Side** with features such as product management, categories, shopping cart, and order processing.

## 🚀 Features

### Customer Features

* User Registration & Login
* Browse Electronic Products
* Product Categories
* Product Search and Filters
* Add to Cart
* Checkout System
* View Order Details

### Admin Features

* Admin Dashboard
* Manage Products (Add / Edit / Delete)
* Manage Categories
* Manage Users
* Manage Orders
* Product Image Upload
